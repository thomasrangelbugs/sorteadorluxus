# Sorteador Luxus

Sorteador Luxus é uma aplicação web criada para realizar sorteios de forma simples, rápida e visual. O projeto permite inserir uma lista de participantes, configurar se nomes repetidos devem contar como chances extras ou não, definir a quantidade de ganhadores e executar o sorteio diretamente pelo navegador.

## Funcionalidades

- Inserção de participantes por lista
- Suporte a um nome por linha
- Opção para considerar nomes repetidos como chances extras
- Opção para remover duplicados e dar apenas uma chance por nome
- Definição da quantidade de ganhadores
- Animação de sorteio
- Exibição do ganhador ou ganhadores
- Interface personalizada para ação promocional Luxus × La Torre
- Layout simples, direto e responsivo

## Objetivo

O objetivo do projeto é facilitar a realização de sorteios promocionais, oferecendo uma ferramenta prática para campanhas, ações comerciais e interações com participantes.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Netlify
